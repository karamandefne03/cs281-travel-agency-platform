# Travel Agency Platform – CS281 Project

## Overview
This project was developed for the CS281 course and simulates a travel agency platform with different user roles and system functionalities.

## User Roles
- Admin
- Customer
- Tour Guide

## Main Features
- Creating and managing tours
- Assigning transportation, hotels, and tour guides
- Searching and purchasing tours
- Viewing purchased tours
- Leaving reviews and ratings
- Capacity control and payment checks

## Files
- `DataSmiths_Final_Stage.ipynb`: main project notebook
- `Functional_Requirements.pdf`: project requirements document

## Technologies Used
- Python
- Jupyter Notebook

## Notes
This project focuses on system logic, user operations, and platform workflow design.
